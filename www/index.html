Kitten.js

A Javascript framework for quick and easy interactivity.

In short, Kitten.js is a tiny Javascript framework optimized for perfomance and size. It is best for adding simple interactivity to static apps, but can also be used to build full scale applications. It is made to be easy to handle, super quick and really tiny, just like a real kitten!
Kitten.js keeps you in the HTML. It takes ideas from Vuejs and Alpinejs, and uses tags that you can add onto components to achieve interactivity. You save state, attributes, even your layout, all in the HTML. Just like it was originally designed.
There are plenty of super massive Javascript frameworks out there, such as React, Vue, Angular, and those are just the most popular ones. If you are looking to easily build an app with a minimal footprint, that still works with large data, you can use this. Kitten.js works best for small features though, like a commenting system, or a simple reactive ui component.
